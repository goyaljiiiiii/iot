# Lessons

These lesson files are small, focused scripts intended for teaching.

Run them from the repo root with the virtualenv activated:

```bash
. .venv/bin/activate
python lessons/lesson_01_open_camera.py
```

## Lesson index

- `lesson_01_open_camera.py`: open a webcam and show frames.
- `lesson_02_count_fingers.py`: MediaPipe hands + simple finger counting.
- `lesson_03_two_finger_screenshot.py`: take a screenshot when 2 fingers are raised.
